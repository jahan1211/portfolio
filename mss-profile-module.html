Design Decisions — Profile Module

Har design decision MSS project document ki requirement se linked hai.

1
Stats Header (Lessons, Streak, Badges)
Profile hero mein 3 stats dikhaye — yeh gamification ke liye zaroori hai. Rural women ko learning continue karne ki motivation milti hai jab woh apni progress dekhti hain.
→ Req 3.3.4: Engaging education module
2
Literacy Level Selector in Edit Profile
User apni reading level khud set kar sakti hai (Icons only / Basic / Full text). Yeh AI ko batata hai ke content kis level par dikhana hai. Bohot zaroori for low-literacy users.
→ Req 3.3.3: AI literacy adaptation
3
Language Screen: Voice Preferences
Sirf language nahi — voice speed (slow/normal), TTS on/off, aur voice input language bhi set hoti hai. Rural users ke liye slow TTS speech bohat helpful hai comprehension ke liye.
→ Req 3.3.3: Voice-based guidance for low-literacy
4
Privacy: Toggles for Each Data Type
Location, usage data, NGO sharing — sab alag alag toggle hain. User control in control deta hai jo trust build karta hai. Rural women ke liye privacy bohot sensitive topic hai.
→ Req 3.3.6: Privacy & data protection module
5
Locked Badges (Grayed Out)
Locked badges dikhana — aur unhe grayed out karna — future goals create karta hai. User dekhti hai ke agle step mein kya milega, jo continuous learning encourage karta hai.
→ Req 3.3.4: Community engagement through learning
6
Emergency Contact Field in Edit Profile
Edit profile mein emergency contact number ka field hai. Yeh SOS module se directly link hoga — jab user SOS bhejegi to is number par bhi alert jayega automatically.
→ Req 3.3.5: Emergency handling & SOS alert system
